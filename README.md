# wanabai.github.io

Static landing page for [wanabai.com](https://wanabai.com). Served by GitHub Pages from the `main` branch root of this repository.

## Content

- `index.html` — the single-page landing site (monospace typography, centered layout, links to the companion articles and the Wanabai and Attestation Layer repositories).
- `CNAME` — custom domain configuration (`wanabai.com`).
- `.nojekyll` — disables Jekyll processing so files are served as-is.

## Local preview

Open `index.html` in a browser. No build step.
